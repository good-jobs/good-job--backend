# good-job-backend
- 굿잡 백엔드 서버 레포지토리
- 사용 기술
  - nestjs / typescript (추후 수정)

## 설치 및 실행

```bash
$ yarn install

# development
$ yarn run start

# watch mode
$ yarn run start:dev

# production mode
$ yarn run start:prod
```

## 테스트

```bash
# unit tests
$ yarn run test

# e2e tests
$ yarn run test:e2e

# test coverage
$ yarn run test:cov
```
